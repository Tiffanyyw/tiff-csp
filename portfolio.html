Build a Human

This project was based off the children dressing games. In the game, you could customize the gender, shirt, pants, shoes and background. You are able to create a bunch of different combinations of styles.

Candy Corn Catcher

In this game, random candy corn and ghost will appear on the screen along with letters. When you type the corresponding letter of a candy corn, your score will go up by 1, but if you click the letter of a ghost, the game will end. You have 30 seconds to catch as many candy corn as you can.

Halloween Card

A halloween card for the children's hospital where they can pick trick or treat. Depending on which option they chose, an image will pop up along with confetti.

A Phishy Fish Tank

This project required us to fix unexpected errores that occured in the alkalinity, temperature, and magnesium files. Each file contained a different code error and once fixed, the program was able to run correctly.

Sharks and Minnows

You will play as a fish and move with the arrow keys. Sharks begin to chase the fish and you attempt to not get eaten. Once in a while tiny fishes will spawn and you can eat them and gain 5 points. Try and stay alive for as long as possible while trying to gain the most amount of points

Command Line GUI

Marshlands

This project required us to locate a lost rover on an alien planet. We were given multiple regions and with the data we were given, we had to determine which region the rover was on. We determined that the region that the rover was on was the Marshlands because the data fit all the components of the Marshlands

AP Project

This is a platformer game made on scratch. There is 3 levels and in order to pass each level you need to obtain the flower to open the door. The goal is to pass all levels in the shortest time.

Honey Production

This project required us to use codes as a way to graph the honey production of all 50 states. We were able to figure out the highest, middle, and smallest honey producers throughout decades, along with the states averages and the US as a whole. This data allows scientists to help honey bee populations as the honey they produce reflects on the health of the colony.
